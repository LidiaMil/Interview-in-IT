# Interview-in-IT
Проект на React предназначен для кандидатов на вакансию и рекрутеров, является агрегатором собеседований с ответами и комментариями.
Каждый кандидат может зарегистрироваться и выложить свое собеседование, а другие пользователи могут написать ответы или прокомментировать вопрос.
Логика приложения также позволяет работать с рейтингом компаний и комментариям.
# Начало работы
1. Для начала нужно установить postgreSQL (https://www.postgresql.org/download/)
 1.1  Заход через терминал под супер пользователем postgres
 sudo su postgres psql или psql -d postgres 
 1.2 Создаем пользователя 
  CREATE USER *имя пользователя* WITH password ' *пароль* ';
 1.3 Создаем  базу данных
  CREATE DATABASE *имя базы данных* OWNER *имя владельца*;
 1.4 Даем привилегии пользователю данной БД 
  GRANT ALL privileges ON DATABASE *название базы данных* TO *имя пользователя*;

3. делаем git clone + скопированный путь в терминале, где открыта нужная папка.
 
4. В терминале открываем папку server и вводим команду npm i  - для того, чтобы все зависимости установились.

5. Заполняем базу данных npx sequelize-cli db:migrate и sequelize-cli db:seed:all 

6. Запускаем сервер npm start

7. В терминале открываем папку client и вводим команду npm i - для того, чтобы все зависимости установились, а далее npm start В этот момент в браузере откроется наше приложение
